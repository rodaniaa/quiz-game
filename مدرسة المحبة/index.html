إعدادات ربط Google Sheets

قم بالربط المباشر لحفظ البيانات والمجاميع تلقائياً في الشيت الخاص بك

📝 خطوات الحصول على رابط الربط:

  1. افتح ملف Google Spreadsheet الخاص بك.
  2. اذهب إلى Extensions -> Apps Script.
  3. امسح الكود القديم والصق كود ملف code.gs المحدث.
  4. اضغط Deploy -> New Deployment.
  5. اختر النوع Web App، واجعل صلاحية الدخول Anyone.
  6. اضغط Deploy وانسخ رابط الـ Web App الناتج والصقه بالأسفل.
